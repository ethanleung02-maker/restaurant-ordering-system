# Restaurant Ordering System (Demo)

A simple Node.js + Express + Socket.io demo for a restaurant ordering system with:
- Customer menu & cart
- Order placement
- Admin order view with real-time updates

## 1. Requirements

- [Node.js](https://nodejs.org/) (version 16+ recommended)
- npm (comes with Node)
